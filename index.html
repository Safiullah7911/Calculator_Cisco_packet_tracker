FLSM vs VLSM Comparison

FLSM (Fixed Length)

All subnets have the same size regardless of requirements.

Example: Divide 192.168.1.0/24 into 4 equal subnets:

  • 192.168.1.0/26 (62 hosts)
  • 192.168.1.64/26 (62 hosts)
  • 192.168.1.128/26 (62 hosts)
  • 192.168.1.192/26 (62 hosts)

‚ùå Problem: Wastes IPs if you need different sizes!

VLSM (Variable Length)

Each subnet sized exactly for its requirements.

Example: 192.168.1.0/24 for different needs:

  • Need 100 hosts ‚Üí /25 (126 hosts)
  • Need 50 hosts ‚Üí /26 (62 hosts)
  • Need 25 hosts ‚Üí /27 (30 hosts)
  • Need 10 hosts ‚Üí /28 (14 hosts)

‚úÖ Benefit: Optimal IP utilization!

Networking Key Concepts

üî¢ Binary & Subnet Masks

Subnet masks use binary to determine network and host portions of an IP address.

IP Address: 192.168.1.10

Binary: 11000000.10101000.00000001.00001010

Subnet Mask: 255.255.255.0 (/24)

Binary: 11111111.11111111.11111111.00000000

Network bits = 1s (24 bits)

Host bits = 0s (8 bits)

üåê Subnetting Fundamentals

Subnetting divides a network into smaller sub-networks for better organization, security, and efficiency.

‚úÖ Benefits:

  • Improved network performance
  • Enhanced security
  • Better IP address management
  • Reduced broadcast traffic
  • Easier troubleshooting

üìã Key Steps:

  • Determine requirements
  • Choose subnet mask
  • Calculate network addresses
  • Define host ranges
  • Document allocation

üì° Network, Host, and Broadcast

üåê Network Address:

First IP in the subnet (all host bits = 0). Identifies the network itself.

üì¢ Broadcast Address:

Last IP in the subnet (all host bits = 1). Used to send to all devices.

üë• Host Addresses:

All IPs between network and broadcast. Assignable to devices.

Example: 192.168.1.0/24

Network: 192.168.1.0

First Host: 192.168.1.1

Last Host: 192.168.1.254

Broadcast: 192.168.1.255

Usable Hosts: 254

üîí Public vs. Private IPs

üåç Public IPs:

  • Globally unique
  • Routable on internet
  • Assigned by ISP
  • Limited availability
  • Requires registration

üè† Private IPs:

  • Used in local networks
  • Not routable on internet
  • Free to use (RFC 1918)
  • Requires NAT for internet
  • Can be reused

üìù Private IP Ranges (RFC 1918):

  • 10.0.0.0/8 ‚Üí 10.0.0.0 - 10.255.255.255
  • 172.16.0.0/12 ‚Üí 172.16.0.0 - 172.31.255.255
  • 192.168.0.0/16 ‚Üí 192.168.0.0 - 192.168.255.255

‚ö° Wildcard Masks

Wildcard masks are the inverse of subnet masks. Used in ACLs and routing protocols.

Formula: Wildcard = 255.255.255.255 - Subnet Mask

Subnet: 255.255.255.0

Wildcard: 0.0.0.255

Subnet: 255.255.252.0

Wildcard: 0.0.3.255

Rule: 0 = must match, 255 = don't care
